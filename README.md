# СТРУКТУРА — мобильное приложение (React Native / Expo)

## Структура проекта

```
struktura-app/
├── App.js                    — точка входа, загрузка шрифтов
├── app.json                  — настройки приложения (имя, ориентация, иконки)
├── constants/
│   └── theme.js               — палитра цветов, отступы, тени, градиенты
├── components/
│   ├── AppBackground.js       — фон: градиентные пятна + металлическая текстура + scanlines
│   ├── MetalButton.js          — кнопка в стиле "металл/стекло" с подсветкой и нажатием
│   └── MetalCard.js            — карточка-контейнер
└── screens/
    └── LoyaltyScreen.js        — первый экран (Лояльность)
```

## Палитра (constants/theme.js)

Все цвета вынесены в один файл — оливковый, синий, зелёный, красный, сиреневый,
плюс базовые тона фона/текста. Совпадает с веб-версией.

## Установка на планшет через Termux (без ПК)

### Шаг 1 — установить Termux
Скачать APK с F-Droid: https://f-droid.org/packages/com.termux/
(не из Google Play — там устаревшая версия)

### Шаг 2 — установить Node.js в Termux
```bash
pkg update && pkg upgrade
pkg install nodejs git
```

### Шаг 3 — скопировать проект
Распакуй архив проекта в папку `~/struktura-app` (через файловый менеджер Termux
или `termux-setup-storage` + перенос из загрузок).

### Шаг 4 — установить зависимости
```bash
cd ~/struktura-app
npm install
npx expo install expo-linear-gradient expo-blur react-native-svg expo-font @expo-google-fonts/anek-devanagari expo-sensors
```

`expo install` сам подберёт версии, совместимые с твоей версией Expo SDK —
поэтому используем именно эту команду, а не обычный `npm install <пакет>`.

### Шаг 5 — установить Expo Go (для тестирования)
Скачать APK напрямую: https://expo.dev/go
(работает без Google Play Services)

### Шаг 6 — запустить проект
```bash
npx expo start
```
Появится QR-код — отсканируй его через Expo Go на этом же планшете
(или открой ссылку `exp://...` прямо в Expo Go).
