
## Шаблон фискального чека — TODO (добавлено 2026-07-21)

**Описание:** Разработка шаблона для печати фискального чека при проведении товара или услуги.

**Что нужно:**
- Дизайн шаблона чека (название бизнеса, ИНН, дата/время, позиции, итог, способ оплаты)
- Генерация чека из данных заказа (order_items + order)
- Вывод через Share API (PDF или текст)
- Возможно — интеграция с Bluetooth-принтером (ESC/POS протокол)
- Отображение QR-кода для электронного чека (опционально)

**Место в плане:** Фаза 8 — Брендинг / Фаза 9 перед публикацией в Google Play

**Связанные данные уже есть:**
- `business_profile`: название, ИНН, адрес, телефон, часы работы
- `orders` + `order_items`: позиции, цены, скидки
- `pay_methods`: тип оплаты (нал/безнал/смешанная)
